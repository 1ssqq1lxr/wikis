## 功能说明

转发数据到log日志中，常用于debug某个设备

## 设计参数

|  参数   | 说明  | 必传  |默认值  |
|  ----  | ----  |----  |----  |
| ruleType  | 规则节点类型 |是 |LOG  |
| script| 脚本 | 否|无  |


## yaml配置

   ```
        smqtt:
            rules: # 规则引擎相关配置
              - ruleName: 过滤
                chain: # 规则链
                  - ruleType: LOG  # LOG
                    script:  '{"topic":"${topic}","msg":"${msg.test}","qos":${qos}}'
   ```
