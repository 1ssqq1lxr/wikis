## 配置详解

|  参数   | 说明  | 必传  |默认值  |
|  ----  | ----  |----  |----  |
| source  | 类型 |是 |RABBIT_MQ  |
| sourceName| 数据源名称 | 是|无  |
| sourceAttributes| 额外参数 | 否|无  |

```markdown
smqtt:
  sources: # 配置数据源sources
    - source: RABBIT_MQ # rabbit mq配置
      sourceName: rabbitmq
      sourceAttributes:
        host: x.x.x.x
        port: 5672
        userName: xx
        passWord: xxx
        queueName: xxx # 队列名称
```

> 支持原生RABBIT_MQ的功能，sourceAttributes设置userName、passWord、queueName.
